Machine Learning | Cybersecurity

Intrusion Detection Algorithmic Efficiency with SMOTE + XGBoost

Proyecto de Machine Learning orientado a deteccion de intrusiones en redes mediante tecnicas de balanceo de datos con SMOTE y modelos XGBoost para mejorar la deteccion de trafico malicioso en datasets desbalanceados.

El proyecto evalua como tecnicas de balanceo de clases pueden impactar el rendimiento de modelos IDS basados en XGBoost, un problema frecuente en ciberseguridad, fraude y anomaly detection.
